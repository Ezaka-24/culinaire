CSS

CSS signifie Feuilles de style en cascade et est un langage utilisé pour décrire la présentation et la mise en forme de documents web, principalement écrits en HTML. Il sépare le contenu (HTML) de son apparence (CSS) et permet aux développeurs de contrôler des aspects tels que la couleur, la police, la mise en page et les animations d'un site web.

Pourquoi CSS?

  • Esthétique : CSS permet de contrôler tous les aspects visuels d'un site, comme les couleurs, les polices, les marges et les arrière-plans, pour le rendre plus attrayant.
  • Cohérence : En définissant des styles une seule fois, CSS assure une apparence cohérente sur toutes les pages du site.
  • Responsive Design : CSS permet d'adapter l'affichage du site à différents appareils (ordinateurs, tablettes, smartphones) et tailles d'écran grâce aux requêtes média.
